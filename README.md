# Telegram Bot Mute

Профессиональный бот для автоматического модерирования новых участников в Telegram-группах. Бот ограничивает права новичков ("мутит") до тех пор, пока они не свяжутся с администратором.

## ✨ Основные функции

- **🛡 Авто-мут:** При вступлении нового участника (самостоятельно или через приглашение) бот автоматически забирает права на отправку сообщений, медиа и ссылок.
- **👋 Приветствие:** Бот отправляет персональное приветствие с кнопкой связи с администратором.
- **🧹 Авто-очистка чата:** Приветственное сообщение автоматически удаляется через 30 секунд.
- **⚪️ White-list (Белый список):** Возможность ограничить работу бота только конкретными чатами для защиты от несанкционированного использования.
- **📊 Умные логи:** Система логирования с ротацией файлов (защита от переполнения диска) и команда удаленного просмотра последних логов.
- **🧠 Защита памяти:** Автоматическая очистка временных данных об обработанных пользователях.

## 🛠 Настройка

Все основные настройки находятся в верхней части файла `Bot_mute.py`:

### Параметры в коде:
- `ALLOWED_CHATS`: Список ID групп, в которых боту разрешено работать. Если пусто `[]` — работает везде.
- `ADMIN_CONTACT_URL`: Прямая ссылка на профиль администратора (для кнопки в чате).
- `WELCOME_MESSAGE_TEXT`: Шаблон сообщения. Можно использовать тег `{full_name}`.

### Переменные окружения (`.env`):
Создайте файл `.env` в корне проекта:
```ini
BOT_TOKEN=ваш_токен_от_botfather
```

## 🚀 Установка и запуск

1. **Клонируйте проект** и перейдите в папку.
2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Настройте `.env`** (добавьте ваш токен).
4. **Запустите бота:**
   ```bash
   python Bot_mute.py
   ```

## 🎮 Команды

| Команда | Описание | Доступ |
| :--- | :--- | :--- |
| `/start` | Включает проверку новичков в текущем чате | Админ группы |
| `/stop` | Выключает проверку в текущем чате | Админ группы |
| `/log` | Отправляет последние 100 строк логов в ЛС | Владелец бота |
