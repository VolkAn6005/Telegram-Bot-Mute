Telegram Bot Mute
Этот бот ограничивает права новых участников в Telegram-чате и отправляет им приветственное сообщение, которое автоматически удаляется через 30 секунд.
Описание
Бот выполняет следующие действия:
Когда новый пользователь вступает в чат, бот ограничивает его права на отправку сообщений.
Отправляет сообщение: Привет, [имя пользователя]! Чтобы иметь возможность писать, напиши администратору любое сообщение.
Через 30 секунд это сообщение автоматически удаляется.
Требования
Python 3.7+
Библиотека python-telegram-bot
Файл .env с токеном бота
Установка
Клонируйте репозиторий:
bash
git clone <URL_репозитория>
cd telegram-bot-mute
Установите зависимости:
bash
pip install -r requirements.txt
Создайте файл .env в корневой папке и добавьте токен вашего бота:
BOT_TOKEN=ваш_токен_от_botfather
Запустите бота:
bash
python Bot_mute.py
Зависимости
Установите их с помощью:
bash
pip install python-telegram-bot python-dotenv
Файлы
Bot_mute.py — основной файл бота.
.env — файл с токеном (не добавляйте в репозиторий!).
